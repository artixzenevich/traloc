# Changelog

## [Unreleased]

### Fixed

- Анимация спиннера и прогресс-бара (вывод в stderr)
- Дублирование файлов при повторном переводе (пропускаются `*.ru.md`)

## [0.1.0] — 2026-07-10

### Added

- Пакетный перевод Markdown-документов через Ollama
- Кэширование переводов и статистика hit/miss
- Маскировка блоков кода и инлайн-кода от перевода
- Разбиение текста на чанки по токенам (с дроблением длинных параграфов)
- Стриминг ответа от LLM со спиннером и прогресс-баром
- Автозапуск Ollama и graceful shutdown с выгрузкой модели
- Флаг `-keep-ollama` для сохранения модели в памяти между запусками
- Флаг `-cache` для указания пути к кэшу
- Флаг `-version` для вывода версии
- Конфигурация через `.traloc.yaml` с вынесенными языками
- Стилизация вывода через lipgloss + charmbracelet/log
- Makefile, golangci-lint, GitHub Actions CI, GoReleaser
- Unit-тесты: cache, code_store, tokenizer
- Фильтр уже переведённых файлов (пропускает `*.ru.md`, `*.de.md` и т.д.)
